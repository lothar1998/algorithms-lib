### Description
Own implementation of popular algorithms and data structures.

### Implemented algorithms:

- ```Bubble sort```
- ```Insertion sort```
- ```Merge sort```
- ```Heap sort```
- ```Quick sort```
- ```Counting sort```
- ```Radix sort```
- ```Bucket sort```
- ```Tree sort```
- ```Maximum subarray```
- ```Selection n-th value of array```


### Implemented data structures:

- ```Stack```
- ```Heap Priority Queue```
- ```Binary Search Tree```
